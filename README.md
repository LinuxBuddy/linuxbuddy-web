# LinuxBuddy

A PHP web application to search Linux commands.

## Deploying

The easiest way to deploy LinuxBuddy is via Heroku.

```sh
$ git clone git@github.com:LinuxBuddy/linuxbuddy-web.git
$ cd linuxbuddy-web
$ heroku create
$ git push heroku master
$ heroku open
```
